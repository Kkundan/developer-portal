Starting with Release v23.11, you can use this REST API to effortlessly create new technical rules.

The `Authorization` must have `Bearer` followed by `Token`.

**Mandatory params:**

- `ruleName`
- `detective`
- `birthright`
- `conditions`
- `actions`
- `removeBirthrightAccess`

**Optional params:**

- `ruleDescription`
- `status`
- `organization`
- `owners`
- `ruletype`
