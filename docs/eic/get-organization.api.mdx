This API returns a list of organizations in SSM.

The `Authorization` must have `Bearer` followed by `Token`.

Optional params:

`organizationname`,
`max`, `offset`

`filtercriteria` - organizationname,organizationtype(possible values -  Data Security, Request Rules, Grouping, 1, 2, 3),status("ACTIVE"/"INACTIVE"),updateuser(username),createuser(username),parentorganization,customproperty&lt;1-20&gt;,description,primarycontact(username),vendormanager(username),location,risk,score

***OR***

`orgQuery`(Example : c.organizationname like '%example_orgname%')
