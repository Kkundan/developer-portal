This API can be used send an email from SSM.

The `Authorization` must have `Bearer` followed by `Token`. 

Note: Configure SMTP in SSM before using this API. 

Mandatory params: 

`to`,
`from`,
`subject`,
`body`

Optional params:

`cc`,`bcc`
