The **uploadSODRuleset** API uses the Ruleset upload file to import rulesets into the User Interface (UI).

- The ruleset upload file exceeding the size of 10MB is not supported.
- Use only .xls or .xlsx file format for the ruleset upload. Do not use any .zip file.

**Mandatory parameters:**

`Upload SOD Ruleset` - Select to browse a (.xls) file for upload. Once the file is uploaded, the name of the file that you attached is displayed.

`existingRuleset` - Specify the value as yes if you are using the existing ruleset.

`rulesetName` - Specify the name of the existing ruleset.

**Other Parameters:**

- `deleteExistingMCAssociations` - Specify the value as yes/no if you wish to delete any existing mitigating controls associations.
- `deleteExistingFunctionAssociations` - Specify the value as yes to delete the existing function associations, else specify the value as No.
- `createmissingEntitlements` - Specify the value as yes to create missing entitlements, else specify the value as No.
