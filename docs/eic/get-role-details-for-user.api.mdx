This API can be used to get the list of all the roles.

This API call can be used to get the list and details of all the roles associated to the user. 

The `Authorization` must have `Bearer` followed by `Token`. 

Optional params:

`username`,
`customproperty<1 to 60>`,
`roletype`,
`requestable`,
`status`,
`role_name`,
`description`,
`displayname`,
`glossary`,
`mininginstance`,
`risk`,
`updateuser`,
`systemid`,
`soxcritical`,
`syscritical`,
`level`,
`priviliged`,
`confidentiality`,
`requestedObject` - contains users and/or entitlement_values
