This API is used to fetch runtime analytics data from SSM.

The `Authorization` must have `Bearer` followed by `Token`.

Mandatory params: 
`analyticsid` ***OR*** `analyticsname`,
`attributes` - runtime control params. 
Sample : For this query - select * from users where manager = $\{manager\} and username = $\{username\}
"attributes":\{
	"manager":"4",
	"username":"johndoe"
\}

Optional params:
`max`,
`offset`,
`loggedinuser`
