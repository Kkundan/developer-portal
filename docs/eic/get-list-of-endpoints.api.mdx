This API returns list of Endpoints in EIC.

The `Authorization` must have `Bearer` followed by `Token`.

Optional params:  
`connectionType`,  
`endpointkey`,  
`endpointname`,  
`max`,  
`offset`,  
`filterCriteria`

**Note**: You must specify the value for `endpointkey` parameter in array. Fr example, `"endpointkey" : [5]`.
