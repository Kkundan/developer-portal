This API is used to create security questions in Saviynt.

The `Authorization` must have `Bearer` followed by `Token`.

Mandatory params:

`question`

`endpointfield`

`section`

Optional Params:

`description`, `questionorder`, `answertype` (by default 'FREETEXT'), `answerrequired` (by default 'false'), `answereditable`(by default 'true'), `defaultvalue`, `updateuser`
